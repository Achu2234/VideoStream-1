# VC Video Player

![logo](https://telegra.ph/file/824cba85b5b820fcd7dcd.jpg)
## Deploy
```
$ git clone https://github.com/Imszy17/tg-stream-video
$ cd tg-stream-video
$ pip install -r requirements.txt
$ python -m lib
```
